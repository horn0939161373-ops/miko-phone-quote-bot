你是誰？

填你自己的名字（方便你之後在 watchlist.json 裡認得是誰）跟你的 LINE User ID（在 LINE Developers Console 的 channel「Basic settings」頁籤可以看到自己的，U 開頭）。記得先把這個 LINE Bot 加為好友，不然收不到推播。

使用步驟

  1. 填你的名字、LINE User ID
  2. 搜尋、勾選想追蹤報價的手機（可複選）
  3. 按「複製設定內容」
  4. 按「在 GitHub 上開啟 watchlist.json」：
    • 如果你是第一個設定的人：把整個檔案內容全選刪除，貼上 [ 剛複製的內容 ]（外面記得加中括號）
    • 如果檔案裡已經有其他人了：在最後一個 } 後面加一個逗號 ,，把剛複製的內容貼在後面、最外層的 ] 之前
  5. 在 GitHub 網頁上按綠色的 Commit changes
  6. 等下一次排程執行（或手動觸發 Actions），有新報價或降價就會推播到你的 LINE
