複製設定內容

複製下面的內容，貼到 repo 裡的 watchlist.json（用下方按鈕可直接開啟 GitHub 網頁編輯器）。

✓ 已複製 在 GitHub 上開啟 watchlist.json →

貼上到 GitHub 的完整步驟

  1. 填你的名字、LINE User ID，搜尋、勾選想追蹤報價的手機（可複選）
  2. 按「複製設定內容」
  3. 按「在 GitHub 上開啟 watchlist.json」：
    • 如果你是第一個設定的人：把整個檔案內容全選刪除，貼上 [ 剛複製的內容 ]（外面記得加中括號）
    • 如果檔案裡已經有其他人了：在最後一個 } 後面加一個逗號 ,，把剛複製的內容貼在後面、最外層的 ] 之前
  4. 在 GitHub 網頁上按綠色的 Commit changes
  5. 等下一次排程執行（或手動觸發 Actions），有新報價或降價就會推播到你的 LINE
