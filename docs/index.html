使用步驟

  1. 在上面搜尋、勾選想追蹤報價的手機（可複選）
  2. 按「複製設定內容」
  3. 按「在 GitHub 上開啟 watchlist.json」，把整個檔案內容全選刪除，貼上剛複製的內容
  4. 在 GitHub 網頁上按綠色的 Commit changes
  5. 等下一次排程執行（或手動觸發 Actions），有新報價或降價就會推播到 LINE
